# clinicBackEnd

This repository contains the back end code for the clinics page application. The application is built using Spring Boot to handle API routes and interactions with a MySQL database.

## Getting Started

These instructions will help you set up and run the project on your local machine for development and testing purposes.

### Prerequisites

- Java JDK installed on your machine.
- MySQL installed and running.

### Installation

1. Clone the repository: https://github.com/PauliusKraujelis/clinicBackEnd.git

2. Import the project into your favorite Java IDE.

3. Configure MySQL database settings in src/resources/application.properties

spring.datasource.url=jdbc:mysql://localhost:3306/{add your dabase name}
spring.datasource.username={your username}
spring.datasource.password={your passwpord}

4. Run the application from your IDE.

The application should now be running.

Pull (securitybasic) branch if you want basic security.
